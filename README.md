# PawCraft

Готовая статическая версия сайта для GitHub Pages и домена **pawcraft.ru**.

## Публикация

1. Загрузите содержимое этой папки в корень GitHub-репозитория.
2. В **Settings → Pages** выберите **Deploy from a branch**, ветку `main` и папку `/ (root)`.
3. В поле **Custom domain** укажите `pawcraft.ru`. Файл `CNAME` уже добавлен.
4. После переключения DNS с Netlify включите **Enforce HTTPS**.

## DNS

Для корневого домена используйте A-записи GitHub Pages, а для `www` — CNAME на `<username>.github.io`. Перед переключением сверьте актуальные значения в документации GitHub.
